Booking widget

Live demo. Drag the bottom-right corner of the frame to resize it and watch the widget respond at real widths (this is a true iframe, so the breakpoints fire exactly as they will in Figma).

Wide → month calendar + info panel · below ~760px → info panel stacks on top · below ~640px → month grid becomes a horizontal day strip.

Embed it on your site

Drop this iframe wherever you want the widget. The small script auto-grows the iframe to fit the widget's height:

<iframe id="rms-frame"
  src="https://rafaelmatosdasilva.github.io/rms-scheduler/embed.html"
  width="1000" height="480" style="border:0;width:100%" scrolling="no" title="Book a time"></iframe>
<script>
  addEventListener("message", function (e) {
    if (e.data && e.data.type === "rmssch:height")
      document.getElementById("rms-frame").style.height = e.data.height + "px";
  });
</script>

In Figma Sites (where you can't add the listener script), append ?fill=1 to the embed URL — the widget stretches to 100% of the iframe box you draw: embed.html?fill=1. Size the frame in Figma and the widget fills it (tall content scrolls inside).

Theme it via URL params (%23 = #):

embed.html?accent=%23ff5c00&bg=%23111827
embed.html?theme=light&bg=%23ffffffCC   (translucent white card)
embed.html?bg=transparent               (see-through, over a dark section)

Params: accent, contrast, dot, bg, surface, fg, muted, border, theme, tz.
