Booking widget

Live demo of the embeddable scheduler. Pick a slot below to book a real event.

Embed it on your site

Copy this snippet anywhere in your page. Replace the endpoint with your deployed Apps Script URL.

<div id="rms-scheduler"></div>
<script
  src="https://YOUR-USERNAME.github.io/rms-scheduler/scheduler.js"
  data-endpoint="https://script.google.com/macros/s/XXXX/exec"
  data-title="Book a time with me"></script>

Re-theme by overriding the CSS variables:

#rms-scheduler {
  --rmssch-accent: #e5484d;
  --rmssch-radius: 16px;
}
